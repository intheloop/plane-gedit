A beautiful theme for Gedit, more information in [Plane project](https://github.com/wfpaisa/plane)

## Install
1. Download in `~/.local/share/gedit/styles/plane-dark.xml`
2. Open: Gedit -> Preferences -> Font & Color | choose: Plane dark
